Online hex calculator performs hex math operations and conversions. Hex addition, hex subtraction, hex multiplication, hex division became fast & easy

[https://www.calculator.io/hex-calculator/](https://www.calculator.io/hex-calculator/)

== Usage ==

Add the Hex Calculator shortcode to your page, post or sidebar:

`[ci_hex_calculator]`

Add the following code to your template where you would like the Hex Calculator to appear:

`<?php display_ci_hex_calculator(); ?>`

== Installation ==

1. Upload the Hex Calculator /ci_hex_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Hex Calculator plugin through the "Plugins" menu in WordPress.
